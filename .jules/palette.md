## 2024-05-08 - Added ARIA Label to Navigation Alerts
**Learning:** For dynamic alert count badges (like the one in LeftNav), screen readers will just read the raw number if unlabelled, which lacks context. Combining an `aria-label` with the surrounding context (e.g., "5 alerts for Hub") improves accessibility. Also, ensuring these small, interactive badges use button styles provides better visual feedback (hover/active states) for mouse and keyboard users alike.
**Action:** Always check numeric badges for accessibility. If they serve as interactive links or buttons, explicitly label them with their purpose and format them with accessible interactive CSS classes.
