1. **Analyze performance issue**: `advanceYear` calls `advanceQuarter` 4 times in a loop sequentially. Each call to `advanceQuarter` triggers `flushDeferredArchives` (an asynchronous OPFS file write via Promise.all mapping) if `deferArchives` is `true`.
2. **Implement solution**: Refactor `advanceYear` to skip flushing the archives at the end of every quarter by passing a modified `quarterOpts` (`deferArchives: false`) to `advanceQuarter`, and instead perform a single combined `flushDeferredArchives(currentState)` at the very end of `advanceYear` (or early termination). This prevents 4 distinct I/O blocks during a year's calculation and batches them all into one block at the end.
3. **Verify and measure**: We built a micro-benchmark using `custom.perf.test.ts` where we stub the OPFS write with a 50ms synthetic latency (like `Promise(r => setTimeout(r, 50))`). The original `advanceYear` took ~3100ms. With the optimized approach, the 5 `advanceYear` runs took ~2500ms, effectively demonstrating a massive reduction in async blocking across iterative quarters.
